## 为什么选择 Musly 作为您的 Navidrome 客户端？

Musly 是 2026 年最佳 Navidrome 客户端，提供以下功能：

- 🎵 **音乐串流** — 从您的 Subsonic 服务器流式播放音乐
- 🎨 **Apple Music 界面** — 以 Apple Music 为灵感的精美现代界面
- 🌙 **深色 / 浅色模式** — 根据系统设置自动切换主题
- 📱 **响应式设计** — 完美适配手机与平板
- 🔍 **全局搜索** — 快速搜索艺术家、专辑和歌曲
- 📚 **音乐库浏览** — 浏览您的完整音乐收藏
- 📋 **播放列表** — 查看与管理服务器及本地播放列表
- ▶️ **正在播放** — 功能完整的音乐播放器，含全套控制
- 🔀 **随机 & 循环** — 灵活控制播放模式
- 📊 **队列管理** — 查看并实时修改播放队列
- 🚗 **Android Auto** — 完整的 Android Auto 集成支持
- 🎧 **同步歌词** — 时间轴同步歌词，桌面端支持 Apple Music 风格全屏模式
- 🧠 **智能推荐** — 个性化混音、「为你推荐」Feed 与收听历史

## 安装与运行

### 前置条件

- Flutter SDK 3.10.0 或更高版本
- 一台兼容 Subsonic API 的音乐服务器（Subsonic、Navidrome、Airsonic 等）

### 步骤

1. 安装依赖：
   ```bash
   flutter pub get
   ```
2. 运行应用：
   ```bash
   flutter run
   ```

### 连接到您的服务器

1. 启动应用
2. 输入您的 Subsonic 服务器 URL（例如：`https://your-server.com`）
3. 输入您的用户名与密码
4. 如使用旧版服务器，请开启「旧版认证」
5. 点击「连接」
